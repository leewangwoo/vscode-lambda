# Lambda AI 코드 어시스턴트 설치 가이드

> 계리팀 계리인프라P 이완우

---

## 1. Visual Studio Code 설치

1. **가젯**에서 `visual studio code` 검색
2. **1.127.0** 버전 설치
3. 나머지는 모두 **다음**을 눌러 설치 완료

---

## 2. Python 3.13.5 설치

1. 설치 화면에서 **"Add python.exe to Path"** 반드시 체크
2. **Install Now** 클릭

> ⚠️ Python 설치가 **완전히 끝난 후**에 아래 설치를 진행해 주세요.

---

## 3. Lambda AI 어시스턴트 설치

1. 아래 `install.zip` 파일을 **`C:\temp`** 폴더에 저장
2. 압축 해제
3. **`install.bat`** 더블클릭 실행

**install.zip**

> 설치 중 보안 경고창이 나오면 **실행**을 클릭해 주세요.
>
> 자동으로 다음 항목들이 설정됩니다:
> - 내부 확장 갤러리 연결
> - 자체 서명 인증서 설치
> - Lambda Chat 확장 프로그램 설치
> - Python 패키지 서버(devpi) 설정

---

## 4. 익스텐션 설정 (채팅 활성화)

> 설치가 완료되면 **반드시 아래 과정을 따라야 AI 채팅이 정상 동작**합니다.

1. VS Code 실행
2. 좌측 사이드바에서 **Extensions** 아이콘 클릭 (또는 `Ctrl + Shift + X`)
3. 검색창에 **`copilot`** 검색
4. **GitHub Copilot Chat**을 찾아 클릭
5. 아래 순서대로 클릭:
   - **Disable AI Feature** 클릭
   - **Reload Window** (`Ctrl + Shift + P` → `Developer: Reload Window`)
   - **Enable AI Feature** 클릭
   - **Reload Window**

---

## 5. 완료 확인

1. `Ctrl + ,` (설정)에서 **`customoai`** 검색
2. 아래 항목이 보이면 성공:
   - **CustomOAI: Url** → `http://100.252.201.200:8088/v1`
   - **CustomOAI: Key** → `dummy-key`

---

## 6. 사용하기

1. 사이드바에서 **Chat** 아이콘 클릭
2. 상단 드롭다운에서 모델 선택 (예: qwen27, qwen35)
3. 자유롭게 질문!

> **예시 질문:**
> - *"파이썬으로 CSV 파일을 읽어서 특정 컬럼만 추출하는 코드를 작성해줘"*
> - *"엑셀 파일에서 특정 시트의 데이터를 읽어 요약 통계를 구하는 코드를 작성해줘"*
> - *"이 파이썬 코드에서 에러가 나는데 수정해줘"*

---

## 💡 자주 묻는 질문

**Q. 설정(Settings)에서 customoai가 보이지 않아요.**

Extensions(`Ctrl + Shift + X`)에서 `copilot` 검색 후 다음 순서로 진행해 주세요:
- **Disable AI Feature** 클릭
- **Reload Window** (`Ctrl + Shift + P` → `Developer: Reload Window`)
- **Enable AI Feature** 클릭
- **Reload Window**

---

**Q. 채팅창에 Lambda가 응답하지 않고 GitHub 로그인을 요구해요.**

Extensions에서 Lambda를 **Disable** 했다가 **Enable** 하고, Reload Window 해주세요.

---

**Q. 설치를 처음부터 다시 하고 싶어요.**

`reset-vscode.bat`을 실행하면 모든 설정이 초기화됩니다. 이후 `install.bat`을 다시 실행하면 됩니다.

---

**Q. Python 패키지(pip install)가 동작하나요?**

네. 설치 시 자동으로 내부 패키지 서버가 설정됩니다. `pip install numpy` 같은 명령이 정상 동작합니다. 단, 필요한 패키지는 미리 서버에 등록되어 있어야 합니다. 필요한 패키지가 있으면 계리인프라P에 요청해 주세요.
